## Getting Started

To run this project locally:

### Step 1: Clone the repo

```bash
git clone https://github.com/charles-254/Cat-App.git
```

### Step 2: Navigate into the project folder

```bash
cd Cat-App
```

### Step 3: Install dependencies

```bash
npm install
```

### Step 4: Run the development server

```bash
npm run dev
```
